# AlphaGenesis - 상위 0.01%급 자동매매 시스템

## 🚀 AlphaGenesis 소개

AlphaGenesis는 상위 0.01%급 성과를 목표로 하는 고급 자동매매 시스템입니다. 머신러닝 모델과 동적 레버리지 시스템을 결합하여 시장의 모든 기회를 포착합니다.

## ✨ 주요 특징

### 🧠 AI 기반 예측 시스템
- **앙상블 ML 모델**: Random Forest, XGBoost, LSTM 결합
- **실시간 학습**: 새로운 데이터로 지속적 모델 업데이트
- **다중 시간대 분석**: 1분~일봉까지 모든 시간대 분석

### ⚡ 동적 레버리지 시스템
- **시장 국면별 조정**: 상승장/하락장/횡보장에 따른 자동 조정
- **Phase별 전략**: Phase1(공격모드) / Phase2(방어모드)
- **리스크 기반 조정**: 낙폭, 연속 손실 등 고려

### 🛡️ 고급 리스크 관리
- **자동 손절매**: 동적 손절매 시스템
- **포지션 사이징**: 리스크 기반 포지션 크기 조정
- **분산 투자**: 다중 코인 분산 투자

## 📊 성능 지표

- **목표 수익률**: 연 300%+ (복리 적용 시)
- **최대 낙폭**: 15% 이하
- **승률**: 65% 이상
- **샤프 비율**: 2.0 이상

## 🚀 시작하기

### 1. 의존성 설치
```bash
pip install -r requirements.txt
```

### 2. 설정 파일 구성
```bash
cp config/trading_config.example.py config/trading_config.py
# API 키 및 설정 수정
```

### 3. 시스템 실행
```bash
# 웹 대시보드 실행
python run_dashboard.py

# ML 백테스트 실행
python run_ml_backtest.py
```

## 📈 Phase별 전략

### Phase1 - 공격 모드 (소액 알트코인)
- **레버리지**: 3-7배
- **대상**: 알트코인 (시가총액 100위 이하)
- **전략**: 변동성 돌파 + 모멘텀

### Phase2 - 방어 모드 (대형 코인)
- **레버리지**: 1.5-5배
- **대상**: 대형 코인 (BTC, ETH, BNB 등)
- **전략**: 평균 회귀 + 트렌드 팔로잉

## 🧠 ML 모델 구성

### 1. Random Forest
- **용도**: 안정적인 가격 예측
- **특징**: 과적합 방지, 해석 가능성

### 2. XGBoost
- **용도**: 고성능 부스팅 예측
- **특징**: 빠른 학습, 높은 정확도

### 3. LSTM
- **용도**: 시계열 패턴 학습
- **특징**: 장기 의존성 학습

### 4. 앙상블
- **용도**: 다중 모델 결합
- **특징**: 안정성과 정확도 향상

## ⚙️ 동적 레버리지 시스템

### 시장 국면별 조정
- **상승장**: +30% (Phase1), +40% (Phase2)
- **하락장**: -40% (Phase1), -30% (Phase2)
- **고변동성**: -50% (Phase1), -40% (Phase2)
- **저변동성**: +20% (Phase1), +30% (Phase2)

### 리스크 기반 조정
- **높은 낙폭 (15%+)**: -40% 레버리지
- **연속 손실 4회+**: -50% 레버리지
- **연속 승리 5회+**: +20% 레버리지

## 🛡️ 리스크 관리

### 1. 자동 손절매
- **동적 손절매**: 시장 변동성에 따른 자동 조정
- **최대 손실**: 거래당 2% 이하
- **일일 손실 한도**: 5% 이하

### 2. 포지션 관리
- **최대 포지션**: 자본의 20% 이하
- **동시 포지션**: 최대 5개
- **레버리지 제한**: Phase별 최대값 준수

### 3. 시장 모니터링
- **거래소 위험 감지**: 파산 위험 시 자동 출금
- **시장 급변 감지**: 급락 시 포지션 청산
- **유동성 모니터링**: 거래량 부족 시 거래 중단
